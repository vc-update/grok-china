1. 通过代理访问 [accounts.google.com/signup](https://accounts.google.com/signup)

1. 访问 [gemini.google.com](https://gemini.google.com)
2. 使用刚注册的 Google 账号登录
3. 同意 Gemini 使用条款
4. 在模型选择器中选择 Gemini 3.1 Pro

## 5、Gemini 3.1 Pro 实用技巧

### 技巧一：充分利用超长上下文

100万 token 是 Gemini 最大的竞争力。实战中：

❌ **不推荐**的方式：
```
"帮我分析这篇论文的第一部分"（等待回复）
"再分析第二部分"（等待回复）
```

✅ **推荐**的方式：
```
"这是三篇相关论文的完整内容（粘贴全文），请帮我：
1. 总结每篇论文的核心观点
2. 对比三篇论文的异同
3. 提出可能的研究方向"
```

一次性提供所有材料，利用超长上下文一步到位。

### 技巧二：多模态 Prompt 模板

图像分析：
```
[上传图片] 
请分析这张图表中的数据趋势，提取关键数字，并给出3条业务建议。
```

视频分析（通过支持视频的平台）：
```
[上传视频]
请总结这段视频的核心内容，列出关键论点，并指出视频中提到的数据或时间节点。
```

### 技巧三：结构化 Prompt 框架

```
角色：你是一位资深的[领域]专家。
任务：请帮我[具体任务]。
要求：
1. [要求1]
2. [要求2]
输出格式：[Markdown表格 / 分点列表 / 完整段落]
```

结构化 Prompt 能显著提升 Gemini 的输出质量，尤其适合长文档处理和专业写作场景。
