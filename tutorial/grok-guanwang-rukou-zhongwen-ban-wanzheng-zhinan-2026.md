## 一、Grok 是什么？为什么值得关注？

Grok 是埃隆·马斯克旗下 **xAI** 公司推出的大语言模型，2023 年首发，2026 年已迭代至 **Grok 4.2**。它的命名来自科幻经典《异乡异客》，寓意"深刻理解"。

Grok 与 ChatGPT、Claude 最大的差异在于以下几点：

| 能力维度 | Grok 4.2 表现 |
|---------|-------------|
| **实时信息** | 直连 X（Twitter）数据流，可获取分钟级实时资讯 |
| **深度推理** | 内置 Think 模式 + Deep Search，多步逻辑推理能力出色 |
| **多模态** | 支持图像理解、文档分析、视频解析 |
| **超长上下文** | 200 万 tokens 上下文窗口，适合长文档处理 |
| **多智能体** | Grok 4.20 Beta 已支持四智能体协作模式 |
| **对话风格** | 不死板、有幽默感，支持个性化调教 |

简单说，**如果你需要一个能实时获取信息、会"思考"、还有点个性的 AI 助手，Grok 是目前最好的选择之一。**

## 二、Grok 官网入口在哪里？

Grok 目前有两个官方入口：

- **xAI 官网**：[x.ai](https://x.ai)
- **Grok 对话入口**：[grok.com](https://grok.com)

### 国内访问现状

遗憾的是，这两个地址在国内 **均无法直接打开**。即便通过科学上网访问，还会面临：

- ❌ 注册需要 **境外手机号**（+86 号码大概率失败）
- ❌ SuperGrok 订阅需要 **国际信用卡**（不支持支付宝/微信）
- ❌ 免费版有严格的 **调用限制**（每 2 小时仅 10 次对话）

**结论：官网不是国内用户的最佳选择。** 下面介绍真正好用的方案。

## 五、Grok 4.2 核心功能详解

### 1. Deep Search（深度搜索）

Grok 的王牌功能。输入一个话题，它会自动搜索数十个信源，交叉验证后生成结构化的深度分析报告。

**实际场景**：
- "分析2026年Q1全球AI芯片市场格局"
- "对比小米SU7 Ultra和特斯拉Model S 2026款"
- "整理最近一周关于GPT-5.4的所有重要消息"

> 详细教程：[Deep Search 深度搜索指南](/tutorial/grok-deepsearch)

### 2. Think 模式（深度推理）

面对复杂问题时，开启 Think 模式，Grok 会展示完整的推理链条。

**擅长的任务**：
- 多步数学推理
- 复杂代码调试
- 逻辑悖论分析
- 策略决策推演

### 3. 多模态理解

支持上传图片、PDF、文档甚至视频文件，Grok 可以直接分析内容：

- 📄 论文 PDF → 一键提取核心观点
- 📊 数据图表 → 自动解读趋势
- 🖼️ 产品截图 → 分析 UI 设计问题
- 🎬 视频文件 → 总结关键内容（GPTCat 支持）

### 4. 实时信息获取

这是 Grok 相比 ChatGPT 最大的差异化优势。Grok 直连 X 平台数据流，能获取 **几分钟内的最新消息**，非常适合：

- 新闻热点追踪
- 舆情监控分析
- 金融市场快讯
- 科技产品发布动态

### 5. 多智能体协作（Grok 4.20 Beta）

xAI 最新发布的 Grok 4.20 Beta 引入了四智能体协作模式，可以让多个 AI "专家"同时协作完成复杂任务。

> 深度解析：[Grok 4.20 多智能体模式实战指南](/tutorial/grok-4-20-multi-agent-china-guide)

- ✅ 中文语义理解精准，无"翻译腔"
