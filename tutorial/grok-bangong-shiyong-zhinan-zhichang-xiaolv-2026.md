## 场景一：邮件撰写与回复

### 痛点
每天花1-2小时写邮件，措辞纠结，生怕语气不对。

### Grok 4.2 实测

Grok写邮件的效果中等偏上。优点是可以联网查背景信息（比如给客户回邮件时自动了解对方公司的最新动态），但中文商务邮件的语气拿捏不如Claude。

### 最佳Prompt模板

```
请帮我写一封商务邮件，具体信息如下：
- 收件人：[对方职位和关系，如"合作方的项目经理"]
- 目的：[邮件目的，如"推迟下周的交付时间"]
- 语气：[正式/半正式/友好]
- 需要包含的要点：
  1. [要点1]
  2. [要点2]
  3. [要点3]
- 字数控制在[X]字以内
- 需要礼貌但不卑微，专业但不生硬
```

### 回复邮件模板

```
以下是我收到的一封邮件，请帮我写一封得体的回复：

[粘贴原始邮件]

回复要求：
- 语气：[同意/婉拒/需要讨论]
- 立场：[你的核心诉求]
- 避免过度承诺
- 保持专业友好
```

### 建议
写邮件最终推荐用Claude 4.6，它对中文商务语境的理解最深。Grok更适合在写邮件前帮你搜索背景信息。

## 场景二：会议纪要整理

### 痛点
开完会要花30分钟整理纪要，经常遗漏重要决议。

### Grok 4.2 表现 🏆

这是Grok的强项场景之一。你只需要把会议中的要点（甚至是杂乱的笔记）扔给它，它就能输出结构化的会议纪要。

### Prompt模板

```
请将以下会议记录整理为正式的会议纪要：

会议信息：
- 日期：[日期]
- 参会人：[人员列表]
- 主题：[会议主题]

我的粗略笔记：
[粘贴你在会议中随手记的内容，不用在意格式]

输出要求：
1. 按议题分类整理
2. 每个议题包含：讨论要点、决议结果、负责人、截止时间
3. 最后附待办事项清单
4. 用表格呈现待办事项
5. 语言简洁正式
```

### 进阶用法
如果你在GPTCat上使用，可以直接上传会议录音（语音转文字），然后让AI整理成纪要，全程自动化。

## 场景三：PPT大纲与内容生成

### 痛点
做PPT最痛苦的不是设计，而是想内容和逻辑。

### 最佳Prompt模板

```
我需要做一个PPT，请帮我生成大纲和每页的内容要点：

PPT信息：
- 主题：[PPT主题]
- 受众：[谁会看这个PPT]
- 目的：[汇报/提案/培训/分享]
- 时长：[演讲时长]
- 页数限制：[期望页数]

要求：
1. 每页标题要有吸引力，不要太平淡
2. 每页列出3-5个要点
3. 标注哪些页面需要配图/图表/数据
4. 给出演讲时的过渡语建议
5. 最后一页要有力的总结
```

### 数据页面的图表建议

```
以下是我PPT中需要展示的数据：
[粘贴数据]

请建议：
1. 最适合的图表类型（柱状图/折线图/饼图等）
2. 图表标题怎么写最有冲击力
3. 需要用颜色标注的关键数据点
4. 一句话的数据解读（放在图表下方）
```

## 场景四：数据分析与报告

### 痛点
Excel数据一大堆，不知道怎么分析、怎么呈现。

### 推荐模型：GPT-5.4

数据分析是GPT-5.4的强项，它可以直接处理数据表格、写分析代码、生成可视化建议。

### Prompt模板

```
以下是我们[部门/项目]的[时间段]数据：
[粘贴数据表格或关键数据点]

请帮我：
1. 分析数据的核心趋势和变化规律
2. 找出异常数据点并给出可能原因
3. 与[上期/同期/行业平均]对比分析
4. 给出3-5条actionable的建议
5. 用适合汇报的语言撰写分析段落（300字以内）
```

### Excel公式辅助

```
我有一个Excel表格，需要实现以下功能：
[描述你的需求，如"根据A列的日期自动统计每月的销售总额"]

请给出：
1. 具体的Excel公式
2. 每个公式的解释
3. 操作步骤
```

## 场景五：周报月报撰写

### 痛点
每周花30分钟写周报，内容重复，写来写去都是那些话。

### 最佳Prompt模板

```
请帮我写一份[周报/月报]：

本周工作内容（流水账版）：
[随便列你做了什么，不用讲究格式]

下周计划：
[随便列下周要做什么]

要求：
1. 把流水账整理成有逻辑的工作总结
2. 突出成果和价值（不是列任务清单）
3. 用量化数据体现工作成果（如果有）
4. 下周计划要体现优先级
5. 语言简洁专业，控制在[X]字以内
6. 加上"需要协调的资源/风险提示"部分
```

### 月度总结模板

```
以下是我本月四周的周报：
[粘贴四周的周报内容]

请整理为月度工作总结：
1. 提炼本月核心成果（3-5条）
2. 量化关键指标的变化
3. 分析本月遇到的挑战和解决方案
4. 下月工作规划和目标
5. 语气要沉稳专业，适合向领导汇报
```

## 场景六：竞品调研与分析

### Grok 4.2 表现 🏆

竞品调研是Grok的王牌场景——Deep Search可以实时搜索竞品的最新动态、产品更新、用户反馈。

### Prompt模板

```
请用Deep Search帮我调研以下竞品：

我们的产品：[你的产品名称和简述]
竞品列表：[竞品1、竞品2、竞品3]

调研维度：
1. 产品功能对比（表格形式）
2. 最近3个月的产品更新动态
3. 用户评价中的高频痛点
4. 定价策略对比
5. 他们做对了什么？我们可以学什么？
6. 他们的弱点在哪？我们的机会是什么？

输出要求：用结构化的报告格式，附信息来源。
```

> 关于Grok Deep Search的详细用法，参考：[Grok Deep Search深度搜索指南](/tutorial/grok-deepsearch)

## 场景七：方案策划

### Prompt模板

```
请帮我撰写一份[项目/活动/营销]方案的框架：

背景：[为什么要做这件事]
目标：[要达成什么效果]
预算范围：[大概的预算]
时间节点：[截止日期]
受众/对象：[面向谁]

要求：
1. 包含目标、策略、执行计划、预算分配、效果评估5个部分
2. 执行计划要有时间轴（甘特图形式的文字描述）
3. 给出2-3个备选方案并对比优劣
4. 预判可能的风险和应对措施
5. 用老板能看懂的语言（不要太多术语）
```

## 场景八：翻译与跨语言沟通

### 商务翻译Prompt

```
请翻译以下商务文档，要求：
- 源语言：[中文/英文]
- 目标语言：[英文/中文]
- 领域：[行业领域]
- 风格：商务正式
- 保留专业术语，术语后用括号附原文
- 注意中英文标点和格式规范的差异

[粘贴原文]
```

### 跨文化沟通建议

```
我需要给[国家/地区]的合作伙伴发一封邮件，
讨论[议题]。请注意：
1. 该文化背景下的商务沟通习惯
2. 需要避免的表达方式
3. 合适的称呼和结尾方式
4. 直接用目标语言写，不要翻译腔
```

## 场景九：头脑风暴与创意激发

### Grok 4.2 表现 🏆

Grok的风格本身就比较跳脱，特别适合做头脑风暴。它不会给你那种"教科书式"的标准答案，而是会给出一些意想不到的角度。

### Prompt模板

```
我们团队正在做[项目/产品]的头脑风暴，主题是[主题]。

已有的想法：
[列出已有想法]

请用发散思维给出15个创意方向：
- 要大胆，可以有5个"疯狂但有启发性"的想法
- 从不同维度切入（用户体验/技术/商业模式/跨界/反常识）
- 每个想法附一句话说明可行性
- 避免和已有想法重复
```

## 场景十：合同与文档审查

### Prompt模板

```
请审查以下合同/文档中的潜在问题：

[粘贴合同文本]

审查维度：
1. 是否存在对我方不利的条款
2. 是否有含糊不清的表述
3. 权利义务是否对等
4. 时间节点和违约条款是否合理
5. 是否有遗漏的常见条款
6. 用表格列出所有潜在风险点及修改建议

注意：我方立场是[甲方/乙方/合作方]。
```

> ⚠️ **重要提醒：** AI审查合同仅供参考，正式签署前务必由专业法律人士审核。

## 效率翻倍的3个秘诀

### 秘诀1：建立Prompt模板库

把你常用的Prompt存起来，下次直接套用，不用每次重新想措辞。

### 秘诀2：多模型配合

同一个任务，先用Grok搜集信息、用GPT做分析、用Claude输出最终文字。三步比一步效果好很多。

在 [GPTCat](https://gptcat.cc) 上可以一个平台切换Grok 4.2、GPT-5.4和Claude 4.6，不需要在多个网站之间跳来跳去。

### 秘诀3：迭代优化

不要期望一次就完美。AI的第一次输出是80分的初稿，你给反馈让它改进，第二次就能到90分以上。

```
刚才的输出有以下问题，请改进：
1. [具体问题1]
2. [具体问题2]
保持其他部分不变，只改进上述问题。
```

## 常见问题

### Q1：用AI写的工作文档，老板能看出来吗？
如果直接复制粘贴，有经验的人可能看出来（过于工整、缺少个人风格）。建议在AI输出的基础上加入自己的观点和表达习惯。

### Q2：办公场景哪个模型性价比最高？
如果只选一个，Claude 4.6在大多数文字类办公场景表现最好。如果需要分析数据，加上GPT-5.4。如果需要查最新资料，加上Grok 4.2。

### Q3：手机上可以用吗？
可以。GPTCat和SnakeGPT都支持手机浏览器直接打开使用，可以把网页添加到主屏幕当App用。开会时用手机打开记录，会后立刻整理纪要。
