## 一、Grok 是什么？为什么值得用？

[Grok 是马斯克旗下 xAI 公司开发的大语言模型](/intro/what-is-grok)，从 2023 年首次亮相到现在已经迭代到 **Grok 4.2** 版本。相比其他主流 AI，Grok 有几个明显的差异化优势：

### 1.1 实时联网能力

Grok 直接接入 X（原 Twitter）的数据流，能获取**分钟级**的实时信息。问它"今天 A 股怎么样"、"刚才那个热搜怎么回事"，它给的答案是真正的"刚刚发生的"，不是几个月前的训练数据。

### 1.2 不设防的回答风格

相比 ChatGPT 和 Claude 经常出现的"我无法回答这个问题"，Grok 的回答更加直接、幽默，甚至有点"毒舌"。这种风格来自马斯克本人的要求——他希望 Grok 是一个"有个性的 AI"。

### 1.3 多模态能力

Grok 4.2 支持文本、图像输入，还能生成图片（基于 Aurora 模型）。你可以扔一张截图给它分析，也可以让它画一张概念图。

### 1.4 超长上下文窗口

132,000 tokens 的上下文窗口，意味着你可以把一整篇论文、一份代码文件丢进去，让它通读后回答问题。

## 二、Grok 官网入口与注册方法（2026年3月实测）

### 2.1 Grok 官网地址

Grok 的官方网站是 **grok.com**，这是 xAI 运营的唯一官方入口。

此外，你还可以通过 X 平台（x.com）左侧导航栏的"Grok"按钮直接进入对话界面。

> 📌 [更多官网入口信息请看这篇](/intro/grok-official-entry)

### 2.2 官网注册步骤

如果你能访问 grok.com，注册流程如下：

**第一步：访问 grok.com，点击"Sign up"**

进入页面后会要求你用 X 账号授权登录。如果没有 X 账号，需要先注册一个。

**第二步：注册 X 账号**

- 访问 x.com，点击注册
- 可以用邮箱注册（Google 邮箱、Outlook 都行）
- **注意**：部分地区需要验证手机号，国内号码可能收不到验证码

**第三步：授权 Grok 访问**

用 X 账号登录后，授权 Grok 访问你的账户信息即可开始使用。

**第四步：选择套餐**

- 免费版：每天有限次数的 Grok 对话（通常 10-20 次）
- Premium+：解锁无限对话和 Grok 4.2 模型

### 2.3 官网注册常见问题

| 问题 | 解决方案 |
|------|----------|
| grok.com 打不开 | 需要科学上网工具，或使用下方的国内平台 |
| X 账号注册收不到验证码 | 尝试用 Google Voice 或其他境外号码 |
| 付款不支持国内银行卡 | 需要 Visa/Mastercard 境外卡，或用下方国内平台 |

## 四、Grok 4.2 实用技巧：让 AI 输出质量翻倍

注册好账号只是第一步，想真正用好 Grok，这几个技巧一定要知道：

### 4.1 用 Deep Search 做深度调研

Grok 的 **Deep Search** 功能会自动搜索多个信息源，交叉验证后给出汇总结果。适合做行业调研、竞品分析等需要综合大量信息的场景。

> 详细教程：[Deep Search 深度搜索指南](/tutorial/grok-deepsearch)

### 4.2 写好 Prompt 的三个原则

好的提示词能让 Grok 的输出质量提升几个档次：

1. **明确角色**：告诉 Grok 它是谁（"你是一位资深的产品经理"）
2. **给出约束**：限定输出格式、字数、风格
3. **提供示例**：给一个你期望的输出样本

> 完整的 Prompt 教程：[Grok Prompt 指南](/tutorial/grok-prompt-guide)

### 4.3 利用长上下文处理文档

Grok 4.2 的 132K 上下文窗口是巨大优势。你可以：

- 把一篇 5 万字的论文粘贴进去，让它做摘要
- 上传代码文件，让它做 Code Review
- 导入会议记录，让它整理行动项

### 4.4 图片生成与分析

在对话框中直接描述你想要的图片，Grok 会调用 Aurora 模型生成。同时，你也可以上传图片让 Grok 分析内容。
