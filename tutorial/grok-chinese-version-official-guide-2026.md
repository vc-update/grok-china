## 国内使用 Grok 的 3 种方法

目前国内用户想要使用 Grok，主要有以下三种途径：

### 方法一：通过镜像聚合平台（推荐）

这是目前**最简单、最稳定**的方式，不需要任何科学上网工具，国内邮箱直接注册即可使用。

镜像聚合平台的原理是通过 API 接入 Grok 官方模型，在国内服务器上提供访问入口，你用到的是和官方一模一样的 Grok 模型，只是访问通道不同。

目前主流的两个平台：

**[SnakeGPT](https://snakegpt.vip)**：运营超过 4 年的老牌平台，支持 GPT-5.4、DeepSeek-V3、Gemini 3.1 Pro、Grok 4.2 等主流模型。国内邮箱即可注册，稳定性很好，适合需要多模型切换的用户。

**[GPTCat](https://gptcat.cc)**：界面一比一还原 ChatGPT 官网，支持 GPT-5.4、Claude 4.6、Gemini 3.1 Pro、Grok 4.2、DeepSeek-V3，还支持 MidJourney 绘图、语音对话和视频对话。模型覆盖最全，体验最接近官方，稳定性极高。

> 💡 **选择建议**：如果你主要用 Grok + GPT，SnakeGPT 足够好用；如果你还需要 Claude、MJ 绘图或者语音视频功能，GPTCat 覆盖更全面。

### 方法二：通过官网直接访问（需科学上网）

如果你有稳定的科学上网条件，可以直接访问 [grok.com](https://grok.com) 使用。

**优点**：直连官方，功能最全，免费用户也有基础额度。

**缺点**：需要翻墙；注册需要海外手机号或 Gmail 等国际邮箱；免费额度有限（每 2 小时 10 次对话）。

详细的注册流程可参考 [Grok 注册教程](/tutorial/grok-register)。

### 方法三：通过 API 接入

适合开发者或有技术基础的用户。xAI 提供了 Grok API，可以集成到自己的应用或工作流中。

API 文档和接入指南详见 [Grok API 教程](/tutorial/grok-api-guide)。

## 常见问题 FAQ

### Q：Grok 中文版是官方的吗？

Grok 本身支持中文对话，但目前没有独立的"中文版"产品。你通过官网或镜像平台使用时，直接用中文提问即可得到中文回答。

### Q：国内用 Grok 需要翻墙吗？

如果直接访问 grok.com，需要翻墙。但通过 [SnakeGPT](https://snakegpt.vip) 或 [GPTCat](https://gptcat.cc) 等镜像平台，**完全不需要翻墙**，国内网络直接使用。

### Q：Grok 免费吗？

Grok 官网有免费额度，但限制较多（每 2 小时 10 次对话）。镜像平台的模式不同，具体以平台页面说明为准。

### Q：Grok 4.2 和之前版本有什么区别？

Grok 4.2 在推理能力、代码生成、多模态理解等方面都有显著提升，尤其是中文语义理解提升了很多。详细测评可看 [Grok 4.2 深度测评](/tutorial/grok-4-2-review-and-setup-guide)。

### Q：手机上能用 Grok 吗？

可以。镜像平台都做了移动端适配，手机浏览器直接访问即可，不需要下载 APP。

### Q：Grok 的数据安全吗？

通过官方或正规镜像平台使用，对话数据都有加密保护。但建议不要在任何 AI 工具中输入极度敏感的个人信息（如身份证号、银行卡号等）。
