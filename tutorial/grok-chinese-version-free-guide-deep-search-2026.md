## Deep Search深度搜索完整教程

Deep Search是Grok最强大的功能之一——让AI像专业研究员一样帮你做全网调研。

### 什么是Deep Search？

开启Deep Search后，Grok不再只依靠自身知识库回答问题，而是：

1. **自动生成搜索关键词**（通常3-8个相关查询）
2. **检索多个信息源**（新闻、论坛、学术文献、社交媒体等）
3. **阅读和分析检索到的内容**
4. **交叉验证信息真实性**
5. **生成结构化的综合报告**

整个过程通常需要30秒到2分钟，取决于问题的复杂度。

### 如何开启Deep Search

**在Grok官网：**
1. 在对话输入框下方找到"Deep Search"按钮
2. 点击开启后输入问题
3. Grok会显示搜索进度（正在检索XX个信息源）

**在GPTCat/SnakeGPT：**
在选择Grok模型后，部分平台支持Deep Search功能。具体操作参考平台指引。

### Deep Search最佳使用场景

#### 场景1：市场调研

```
用Deep Search调研2026年国内低代码平台市场，给出：
1. 市场规模和Top 5玩家
2. 各平台核心差异
3. 用户评价汇总
4. 发展趋势预测
```

Grok会自动搜索行业报告、用户论坛、产品官网，给出一份全面的调研报告。

#### 场景2：竞品分析

```
Deep Search对比Notion和Obsidian在2026年的功能差异、用户口碑、定价策略
```

#### 场景3：技术选型

```
我要开发一个AI聊天应用，用Deep Search对比LangChain vs LlamaIndex vs Semantic Kernel，从易用性、性能、社区活跃度三个维度分析
```

#### 场景4：投资研究

```
用Deep Search分析[公司]的最新财报、市场表现、分析师评级和潜在风险
```

#### 场景5：学术调研

```
Deep Search搜索"大语言模型幻觉问题"的最新研究进展（2025-2026），总结Top 5论文的方法和结论
```

### Deep Search使用技巧

1. **问题要具体**：越具体的问题，Deep Search的检索效率越高
2. **加时间限定**：加上"2026年""最近3个月"等限定词，确保获取最新信息
3. **要求结构化输出**：在Prompt中指定输出格式（如"分为优势、劣势、建议三个部分"）
4. **多维度对比**：Deep Search擅长多维度横向对比，善用这个能力
